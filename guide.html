Panduan Lengkap Penggunaan VPS Tunneling

Ikuti panduan langkah demi langkah di bawah ini sesuai dengan metode koneksi yang Anda inginkan.

A. Persiapan:

  1. Pastikan Anda sudah menerima detail akun (Username, Password, IP Server) dari kami.
  2. Download dan install aplikasi **Bitvise SSH Client** (untuk Windows) atau **Termius** (untuk Android/iOS/Mac).

B. Langkah Koneksi:

1. Buka aplikasi SSH Client Anda.

2. Masukkan **Host** (IP Server Anda), **Port** (biasanya 22 atau 443), dan **Username** Anda.

3. Di bagian 'Login', masukkan **Password** yang sudah diberikan.

4. Klik tombol **Connect/Login**. Selesai!

Catatan: Untuk menggunakan fitur SSH Tunneling/Proxy, pastikan Anda mengatur SOCKS proxy di browser/aplikasi Anda ke **localhost:Port_Lokal** (Port lokal default Bitvise adalah 1080).

A. Persiapan:

  1. Download aplikasi **OpenVPN Connect** dari Play Store/App Store.
  2. Download file konfigurasi (file `.ovpn`) dari Area Pelanggan kami (link akan kami berikan).

B. Langkah Koneksi:

1. Buka OpenVPN Connect, lalu pilih **Import Profile**.

2. Pilih file `.ovpn` yang sudah Anda download.

3. Masukkan **Username** dan **Password** akun Anda saat diminta.

4. Geser tombol **Connect**. Jika berhasil, ikon kunci akan muncul di status bar Anda.

Panduan ini membutuhkan file konfigurasi WireGuard (`.conf`).

1. Install aplikasi WireGuard di perangkat Anda.

2. Import file konfigurasi yang didapatkan dari Area Pelanggan.

3. Tekan tombol **Aktifkan**. Selesai!
