Layanan Kami

Pilih protokol dan server. Akun akan dibuat otomatis setelah pembayaran terverifikasi.

Cara Kerja

  1. Pilih layanan & server → klik Beli.
  2. Selesaikan pembayaran (Pakasir / QRIS / gateway lain).
  3. Webhook menerima notifikasi → backend membuat akun otomatis.
  4. Kamu menerima konfigurasi via Telegram / email.

FAQ

Apa metode pembayaran yang didukung?

Kami mendukung Pakasir (QRIS), transfer, dan metode lain yang diaktifkan admin.

Bagaimana saya menerima akun?

Setelah pembayaran terverifikasi, akun dikirim otomatis ke Telegram jika kamu berikan Telegram ID saat checkout, atau admin akan mengirim melalui email.
