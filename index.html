Cara Kerja

  1. Pilih layanan & server → klik Beli.
  2. Selesaikan pembayaran.
  3. Server memproses pembayaran (otomatis via webhook).
  4. Setelah pembayaran konfirmasi, akun dibuat dan dikirim otomatis.

FAQ

Apa yang saya dapat setelah bayar?

Setelah pembayaran berhasil: username/password (untuk SSH) atau file konfigurasi/link (untuk VMESS/VLESS/TROJAN) akan dikirim otomatis melalui Telegram atau email.

Bagaimana metode pembayaran?

Kami menggunakan gateway pembayaran yang mendukung QRIS dan transfer bank. (Detail metode disediakan saat checkout)

Apakah ada garansi?

Kami menyediakan support teknis; untuk masalah server akan ditangani oleh admin. Kebijakan refund mengikuti ketentuan toko.
